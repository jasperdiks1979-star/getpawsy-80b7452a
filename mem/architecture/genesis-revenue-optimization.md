# Genesis Revenue Optimization Engine (ROE)

Commercial optimization layer. Profit-first, sustainability-first.

## Tables (roe_*)
- **roe_snapshots** — daily unified commercial model (visitors → orders → revenue → margin → cash flow + CAC/LTV/payback/ROAS).
- **roe_revenue_tree** — node values with sensitivity ∂revenue/∂node and ∂profit/∂node.
- **roe_bottlenecks** — top constraints across 8 areas with expected unlock $ and recommended actions.
- **roe_marginal_value** — elasticity-driven expected impact per lever (CTR/CVR/AOV/repeat/refund/CAC/price/bundle).
- **roe_portfolio_ranks** — per-product composite scored via `roe_portfolio_score()` (profit + margin + growth + trend + LTV + inventory − refund − ops − shipping) → scale/hold/optimize/retire.
- **roe_capital_allocations** — recommended % split of engineering/AI credits/publish/video/creative/ads/SEO/expansion/infra.
- **roe_simulations** — what-if scenarios with revenue/profit/cash/risk + confidence.
- **roe_forecasts** — daily/weekly/monthly/quarterly/annual revenue + profit with 95% CIs.
- **roe_unit_economics** — contribution margin, break-even CPA & ROAS, LTV:CAC, payback, operating margin.
- **roe_customer_value** — per-segment expected LTV/repeat/profit/referral/churn.
- **roe_pricing_recommendations** — approval-gated (pending_approval → approved/rejected/applied).
- **roe_scaling_opportunities** — channel × target with expected marginal return.
- **roe_executive_scorecard** — composite revenue/profit/growth/efficiency/margin/capital/forecast/scaling/business value scores, generated by `roe_compose_scorecard()`.
- **roe_consultations** — audit of every engine call.

## API — supabase/functions/roe-api (Lovable AI, `google/gemini-3-flash-preview`)
`ingestSnapshot`, `recomputeRevenueTree`, `findBottleneck`, `marginalValue`, `rankPortfolio`, `recommendInvestment`, `simulate`, `predictRevenue`, `recordUnitEconomics`, `recommendPricing`, `approvePricing`, `recommendScaling`, `calculateEnterpriseValue`, `searchRevenueKnowledge`, `stats`. Every call writes `roe_consultations`.

## Governance
- Recommendations only. ROE never executes operational changes.
- Pricing recommendations require explicit human approval (`approvePricing`).
- Forecasts always carry 95% confidence intervals.
- Optimizes for sustainable long-term enterprise value, not vanity metrics.

## Client / UI
`src/lib/roe/client.ts` exports `ROE`. Dashboard at `/admin/revenue-optimization` with Revenue Tree, Bottlenecks, Portfolio, Capital, Simulate, Forecasts, Pricing, Scaling, Search tabs.
